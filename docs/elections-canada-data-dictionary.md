# Official Voting Results (2025) - Format 2

The "pollresults_resultatsbureau" format provides an entry for each candidate by polling station in each electoral district. It includes information on who the candidate is, the candidate's political party, and the voting results for that candidate at a particular poll.

| **Field Name** | **Description** |
| Electoral District Number | The electoral district number. |
| Electoral District Name_English | The English name of the electoral district. |
| Electoral District Name_French | The French name of the electoral district. |
| Polling Division Number | The number assigned to the polling division, or subsection thereof. For example: 3, 45A, 48-3, 601. |
| Polling Division Name | A name that generally represents the locality of the polling division boundary. |
| Void Poll Indicator | Indicates that a poll exists but has no electors. |
| No Poll Held Indicator | Indicates that the returning officer intended to hold this poll, but unforeseen circumstances prevented it. |
| Combined with No. | "Combined" polling divisions are those where electors from different polling divisions placed their ballots in the same ballot box so that service to electors would be more efficient. This term may also be used in cases where votes from two ballot boxes were counted together; or where all electors in one poll voted for the same candidate and their ballots were reported combined with another poll in order to maintain the secrecy of the vote. |
| Rejected Ballots for Polling Station | The number of rejected ballots at this poll. |
| Electors for Poll | The number of electors on the list of electors for this polling. |
| Candidate's Family Name | The family name of the candidate. |
| Candidate's Middle Name | The middle name of the candidate. |
| Candidate's First Name | The first name of the candidate. |
| Political Affiliation Name_English | The short-form English name of the candidate's political affiliation. |
| Political Affiliation Name_French | The short-form French name of the candidate's political affiliation. |
| Incumbent Indicator | "Y" if candidate was the incumbent, "N" otherwise. |
| Elected Candidate Indicator | "Y" if candidate was elected, "N" otherwise. |
| Candidate VoteCount | The number of valid votes the candidate obtained at this poll. |
